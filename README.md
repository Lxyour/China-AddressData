# China-AddressData
中国最新行政区域划分数据

## File index
* province.json                   ------ 仅包含省份和省级简称
* provinceCityList.json           ------ 包含省级和城市区号
* /three-level address           
  * addrData.json           ------ 包含省市区
  * province.js             ------ 省份
  * city.js                 ------ 城市
  * district.js             ------ 县区
